HOW TO CREATE AN ETALK USING THE DOCKER CONTAINER

The following etalk guide is suitable for educational purposes. It permits for the use and testing of the etalk application on a user's own platform. This guide uses the docker-compose tool to set up the application services (etalk php/apache, mysql, phpmyadmin). These guidelines are addressed to people without specific computing knowledge.

REQUIREMENTS

MAC : Requires Apple macOS Yosemite 10.10.3 or above

WINDOWS : Requires Microsoft Windows 10 Professional or Enterprise 64-bit

Browsers: Google Chrome; Safari (version ≥7); Internet Explorer (version 11).

I. INSTALL DOCKER and run the eTalk virtual machine (VM)

Follow these guidelines; short videos will help you if necessary.

  1. Go to the URL: https://github.com/docker/toolbox/releases/tag/v1.12.5

And download the file: DockerToolbox-1.12.5.pkg

  1. Start the step by step installation by double clicking the .pkg format . When this window appears: 

Click on the “continue” or “next” button: the installation will be finished.

3. Then download or clone the repository https://github.com/msank/etalk-docker by clicking on  

And save the file to the Desktop folder by clicking on the “download zip” button : 

4. Copy the folder named ‘etalk-docker-master’ to the Desktop, or unzip it if necessary (double-click on the zip file ‘etalk-docker-master.zip’). It should create a folder named ‘etalk-docker-master’ on the Desktop.

5. If you have a Windows or MacOSX operating system, open the application named "docker quick start terminal" (you can find it by opening the applications menu, and then clicking on the docker application icon).

If you have a Linux operating system, open a terminal .

In Windows or MacOSX, such a window, called a shell, automatically opens. Please wait until you see this smart whale:

6. On the command line, in the shell, go to the Desktop directory by typing or pasting the following command after the $ symbol:

cd Desktop/etalk-docker-master

Then execute the command by clicking on the ‘return’ or ‘enter’ key on your keyboard.

7. Secondly, ‘build’ the image by typing or pasting the following command (after the $):

docker-compose build

Then execute the command by clicking on the ‘return’ or ‘enter’ key on your keyboard and wait until it is finished.

8. Then, run the etalk application by pasting the following command after the $:

docker-compose up -d

Then execute the command by clicking on the ‘return’ or ‘enter’ key on your keyboard and wait until it is finished. It can take a few moments.

9. Open a browser (e.g Safari, Chrome or Internet explorer) : 

In Windows or Mac OSX: go to the URL http://192.168.99.100:88 by typing it into the address bar and pressing ‘enter’ for the viewer interface or use the address http://192.168.99.100:88/edit for the edit interface.

In Linux: go to the URL http://localhost:88/ for the viewer interface or http://localhost:88/edit for the edit interface.

Your browser window should display the following as the viewer interface:

And see this window for the edit interface :

II. SECOND STEP: MAKE YOUR OWN ETALK

To make your own eTalk, you can follow the “HOW-TO” eTalk "Make your own eTalk" by accessing the following address http://192.168.99.100:88

  1. In the “data” folder, create and name a new folder that will contain the mp3 files. You can find the “data” folder by opening the “etalk-docker-master” folder on your computer, then open the “etalkapp” folder, then the “etalk-master” folder, and then finally the “data” folder.
  2. Go to the edit interface  http://192.168.99.100:88/edit, and start editing your eTalk (page “How to do your own eTalk”).

************************************************************************************************************************

III. RUN the application next time:

1. If you have already completed the installation (see Section I above), you can simply run the etalk-docker by reopening a “quick start terminal” (to find it, go to the Applications menu, then select the docker application icon). First wait until you get the whale image (see point 5 above). Then paste the following command into the command line (after the $): 

cd Desktop/etalk-docker-master

And click on the ‘enter’ or ‘return’ key on your keyboard.

2. Then paste the following command (after the $):

docker-compose start

And click on the ‘enter’ or ‘return’ key on your keyboard.

3. Then open a browser (e.g Safari, Chrome or Internet explorer): 

In Windows or Mac OSX : go to the URL http://192.168.99.100:88 for the viewer interface or http://192.168.99.100:88/edit for the edit interface.

In Linux : go to the URL http://localhost:88/ for the viewer interface or http://localhost:88/edit for the edit interface.

2. Q: The docker-compose build command is taking too much and I think it is stalled. What should I do? 

A: You should not wait more than a few minutes. If it takes too long, please don’t hesitate to restart your computer.
